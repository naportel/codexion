# Description

Codexion is program written in C that simulates a "table of coders" which each coder has one "dongle" on their left, but they need two dongles to work, so they have to temporarily borrow a dongle from the next coder to do their routine until it "compiles" and it goes to the next coder from a certain order "fifo" (first in, first out) or "edf" (Earliest Deadline First), and it goes on and on until one of them takes too long that they burnout and the program ends.
